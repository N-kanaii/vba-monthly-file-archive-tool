# vba-monthly-file-archive-tool
Automates monthly file organization: archiving past files, renaming current files, and managing folder structure using Excel VBA.

## Overview
Automates monthly file organization using Excel VBA.

This tool performs:
- Archiving past files into year/month folders
- Renaming current files
- Managing folder structure automatically

## 概要
Excel VBAを使用して、月次ファイルの整理を自動化します。

- 過去ファイルの年度・月別アーカイブ
- 当月ファイルのリネーム
- フォルダ構成の自動管理

## Files
- vba-monthly-file-archive-tool.xlsm : Excel macro-enabled file
- monthly-file-archive.bas : VBA source code

## Technology
- Excel VBA
- FileSystemObject
